## 🎯 Featured Project: Smart Office Appointment System

### 📋 Project Overview

A full-stack appointment management system built with **Next.js** and **NestJS** that helps offices manage appointments efficiently.

### ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🔐 JWT Authentication | Secure login and registration |
| 👥 Role-Based Access | Admin, Staff, and User roles |
| 📅 Appointment Booking | Real-time availability checking |
| ✅ Staff Management | Approve/reject with comments |
| 📊 Admin Dashboard | Analytics and charts |
| 📎 Export to Excel | Download appointment reports |
| 🔔 Real-time Notifications | Status updates |
| 📱 Responsive Design | Works on all devices |
